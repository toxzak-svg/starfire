## Key Findings (from research)

### ✅ Viable on CPU
- **SQA solver** — genuine optimization advantage for non-convex Ising problems, ~1min per run
- **ESN** — echo state networks, proven reservoir computing, NumPy implementation ~10ms/1000 steps
- **Global Workspace** — tractable consciousness architecture, straightforward to implement
- **RQA metrics** — recurrence quantification for consciousness proxies, ~0.1s for N=1000
- **Lyapunov monitoring** — real-time chaos detection, ~0.5s for N=100
- **Tensor network compression** — 10-100x compression of cognitive states achievable

### ⚠️ Tractable But Novel
- **Creative oscillation controller** — no existing implementation, needs empirical validation
- **Φ (Integrated Information)** — exact calculation intractable for N>20; use proxy metrics instead
- **Quantum-inspired autoencoder** — drop "quantum" framing; use established vector symbolic architectures

### ❌ Not Feasible
- Exact Φ calculation for large systems (exponential complexity)
- "Quantum speedup" claims without actual quantum hardware
- Full AGI integration in <12 months (honest assessment)
