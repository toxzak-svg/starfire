# Component Crate

**Never hunt for that modal again.**

A VS Code extension for saving, searching, and reusing your Tailwind + React components. Import from GitHub, preview in-editor, and insert anywhere.

## Features

- **🔍 Search Components** — `Ctrl+Shift+P` → "Search Components" to find anything you've saved
- **💾 Save Selection** — Select code → Right-click → "Save to Crate"
- **⚡ Quick Insert** — Insert any saved component at your cursor in one click
- **📥 GitHub Import** — Browse any GitHub repo and import components directly
- **👁️ Live Preview** — Preview any saved component's full code
- **📊 Status Bar** — Always see how many components you've saved

## Getting Started

1. Install from VS Code Marketplace (coming soon)
2. Open VS Code and press `Ctrl+Shift+P`
3. Type "Save to Crate" after selecting some code
4. Or press `Ctrl+Shift+P` → "Search Components" to browse

## Commands

| Command | Shortcut | Description |
|---------|----------|-------------|
| Search Components | `Ctrl+Shift+P` → "Search Components" | Find saved components |
| Save to Crate | Right-click selected code | Save current selection |
| Insert Component | `Ctrl+Shift+P` → "Insert Component" | Insert from saved |
| Import from GitHub | `Ctrl+Shift+P` → "Import from GitHub" | Pull from any repo |

## GitHub Integration

1. `Ctrl+Shift+P` → "Import from GitHub"
2. Browse your repos (or any public repo)
3. Pick files → they import instantly
4. Components stay linked to their GitHub source
