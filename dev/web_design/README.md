# Web Design & Aesthetics Projects

This folder contains all web design, webpage generation, and aesthetics-related experiments and data.

## Contents

### Datasets
- **website-aesthetics-datasets/** - Comparison-based and rating-based aesthetics datasets
- **webpage_data_collection/** - Web page data collection scripts and utilities

### Models & Experiments
- **webpage_models/** - Trained models for webpage-related tasks
- **train_image_vae_aesthetics.py** - Training script for image VAE on aesthetics data
- **validate_image_vae.py** - Validation script for image VAE models
- **image_vae_aesthetics/** - Training results for image VAE on aesthetics data
- **image_vae_validation/** - Validation results for image VAE models

### Integration Projects
- **demo_app_evolution/** - Demo application evolution experiments
- **test_webalchemist_integration.py** - WebAlchemist integration tests

### Documentation
- **AI_APP_BUILDER_DESIGN.md** - Design document for AI app builder
- **WEBALCHEMIST_INTEGRATION.md** - WebAlchemist integration documentation
- **WEBPAGE_BUILDER_INTEGRATION_PLAN.md** - Integration plan for webpage builder

## Purpose

These projects explore aesthetic understanding, webpage generation, and visual design through:
- VAE models trained on aesthetics datasets
- Integration with WebAlchemist for webpage generation
- Collection and analysis of webpage design patterns
